# StudyMate AI

An AI-powered full-stack web application that enables users to have intelligent conversations with multiple AI models in a modern, responsive interface.

## Overview

StudyMate AI is designed to provide a fast, intuitive, and feature-rich AI chat experience. The application focuses on clean UI, secure authentication, real-time streaming responses, and an efficient user experience.

---

## Features

* AI-powered conversations
* Real-time streaming responses
* Secure user authentication
* Chat history management
* Responsive design
* Dark and Light mode
* Markdown support
* Code syntax highlighting
* Mobile-friendly interface

## Installation

```bash
git clone <repository-url>

cd studymate-ai

npm install

npm run dev
```

---

## Environment Variables

Create a `.env.local` file.

```env
OPENAI_API_KEY=your_api_key
DATABASE_URL=your_database_url
```
